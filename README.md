 Dudu IVHub
Plan Your Industrial Visit Without the Stress
🌍 Overview


Dudu IVHub is a streamlined platform designed to simplify industrial visit planning for students.
It removes the complexity of coordinating with companies, arranging travel, and managing logistics — all through a single, structured flow.

🎯 Core Concept

You book. We plan. You experience.

⚡ What This Platform Does

Dudu IVHub handles the complete industrial visit lifecycle:

🏢 Industry connection & approval
🚌 Travel planning & coordination
🍽️ Food & accommodation arrangements
📍 Pickup and safe return
📋 Structured execution from start to finish
📌 Booking System (Google Form Integration)

The platform uses a Google Form-based booking system to collect and process requests.

📝 How Booking Works
User clicks “Book Now” on the website
Redirects to a Google Form
User submits:
Name
College
Visit requirements
Additional details
Submission is received and verified
Process begins internally

This approach ensures:

Simple user experience
Fast data collection
Easy management of requests

💬 Feedback System

A dedicated feedback form (Formspree integration) is included on the website.

Purpose:

Collect real student experiences
Improve service quality
Build trust and credibility
Data Collected:
Name
College
Industrial visit experience

🧭 Workflow

The system follows a structured 5-step process:

Booking Received
User submits request via Google Form
Company Confirmation
Industry approval and coordination
Planning & Coordination
Scheduling, logistics, and flow design
Execution
Travel, food, and stay arrangements
Completion
Safe pickup and drop
🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript
Booking System: Google Forms
Feedback System: Formspree
UI/UX: Custom animations, responsive design
Fonts: Google Fonts
